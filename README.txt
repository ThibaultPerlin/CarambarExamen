La page d'accueil contient un bouton pour afficher une blague aléatoire.
Apuuyer dessus.
Vous pouvez voir la blague s'afficher et son numéro dans la zone de texte juste en dessous.
Vous pouvez aussi voir plus d'information sur ces blagues dans la console en appuyant sur F12.
Vous pourrez donc voir un tableau qui dispose de toutes les blagues.
Mais aussi de voir en temps réel les informations d'une blague apparu une fois que le bouton a été actionné afin de voir la blague en question et son ID dans la console.
